Hexo博客框架

很多人都希望有自己的博客或个人主页，但又觉得新浪博客等平台太丑，而大费周章搞独立服务器太贵太麻烦。那么，GitHub、Gitee等代码托管平台的静态pages是个很好的选择。其中Github支持免费自定义域名，优先推荐；Gitee的自定义域名需要收费，但访问速度上极具优势。

笔者的博客托管在Gitee上，主页托管在GitHub上。

但这些平台的本意是代码托管，所以只支持静态网页，不支持服务端操作，因此诸多静态网页博客系统应运而生，Hexo就是其中之一。

概述

hexo是一款基于node.js的博客框架，使用markdown解析文章，可以快速生成静态网页。
